Decimal Time

This is my decimal time project (Using EST):

What's so different about decimal time? Well, it uses multiples of 10s instead of 12s. For instance:

  • Each day is split into 2 halfs, with each half being 10 decimal hours long
  • Each hour is 100 decimal minutes long
  • Each minute is 10 decimal seconds long

A decimal second is about 4.32 regular seconds, so the decimal second can still be used for counting time. A day ends up being 20000 decimal seconds, a nice number, instead of 86400 regular seconds.
